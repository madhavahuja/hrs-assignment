Automation


Replace the following values in jenkinsfile

<AWS_ACCESS_KEY>\n
<AWS_SECRET_KEY>
<REGION>
<ACCOUNT_ID>
<REPOSITORY_NAME>
<CLUSTER_NAME>
<DEPLOYMENT_NAME>
<CONTAINER_NAME>
<NAMESPACE>

with the exact values. Theese are the placeholders added for your understanding and better clarity.
